# Energy Consumption Prediction made by AKSHAT KUMAR SINGH

**Energy Consumption** is a machine learning model that predicts the electricity bill for a month based on input parameters such as the number of rooms, people in the house, and electric appliances. The project utilizes Flask, NumPy, Pandas, and Matplotlib for data processing, prediction, and visualization.

## Overview

- **Project Name:** Energy Consumption Prediction

## Features

- **Electricity Bill Prediction:** Predict the monthly electricity bill based on input parameters.
- **Interactive Visualization:** Displays a graph comparing the predicted and actual energy consumption data.
- **Web Interface:** User-friendly HTML form for input and results display.

## Technologies Used

- **Backend Framework:** Flask
- **Data Handling:** Pandas, NumPy
- **Machine Learning Model:** Linear Regression
- **Visualization:** Matplotlib
- **Frontend:** HTML, Bootstrap
